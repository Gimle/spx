Welcome to Gimlé Spx Extension
===============================

Under development.


Requirements
------------
php 5.3 / 5.4


Dependencies
------------
Common extension.


Installation
------------
Download or checkout from reposirtory to a non-public folder.

Add this to the core ini file to load globaly, or to a project ini file to load for that project only.
spx = "/absoulute/path/to/gimle/spx/extension/"

* This extension must be loaded after common extension.
